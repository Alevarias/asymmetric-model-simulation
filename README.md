# Introduction
This project simulates walking skeletal models under asymmetric weight loads to explore the different compensation methods optimal control can reveal.

# Requirements
- Python 3.11.14
- Conda

# Installation
1. [Setting Up Conda OpenSim Package](https://opensimconfluence.atlassian.net/wiki/spaces/OpenSim/pages/53085346/Scripting+in+Python)
```bash
$ conda create -n opensim_scripting python=3.11 numpy
$ conda activate opensim_scripting
$ conda install -c opensim-org opensim-moco
```
2. Installing Other Dependencies
```bash
pip install -r requirements.txt
```
